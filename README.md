# Team_Tracker

## Description

* This project allows for a business owner the ability to have full control over their employee database.  Users have the ability to read and update each employee's brief profile for every department.

## Installation
* To install, clone the repo and then run "npm install inquire" in your terminal.  Afterward run the "node server" in the command and answer the prompts
